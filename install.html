HTML Report Viewer

强大的 HTML 仪表板查看器，让您在 Excel Online 和桌面版中轻松查看和交互 HTML 报告。

🚀 一键安装到 Excel Online

🌟 主要功能

  • 在 Excel 中直接查看 HTML 仪表板
  • 支持交互式图表和动态内容
  • 兼容 Excel Online 和桌面版
  • 共享运行时，性能优异
  • 安全的 HTTPS 连接
📝 注意： 需要 Microsoft 365 账户登录。安装后插件将出现在 Excel 的"插入"菜单中。

🔧 手动安装方式

如果自动安装不工作，请按照以下步骤手动安装：

  1. 打开 Excel Online
  2. 创建新工作簿或打开现有工作簿
  3. 点击 插入 → Office 加载项
  4. 选择 上传我的加载项
  5. 上传此文件：manifest_online.xml
